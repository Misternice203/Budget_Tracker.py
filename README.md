### Budget_Tracker.py
A simple Command line budget tracker application built using Python, This project allows users to record income and expenses, view transaction history,  and save info using persistent storage.

## This project is actively being improved as part of my backend developer journey.

## Features
* Add income transactions
* Add expense transactions
* View current balance
* View all transactions
* Persistent storage using JSON
* Simple and user friendly CLI menu

## Concepts
This project demonstrates:
* Object Oriented Programming (OOP)
* Class design and responsiblity seperation
* File handling with JSON
* Data persistence
* Basic error handling
* CLI app sturcture

## Project Structure
* Transaction -> Represents a single income or expense
* Budget -> Stores and manages transactions
* Report -> Calculates totals and balance
* Storage -> Handles saving/loading data to JSON (file is labeled data.json)
* App -> Controls user interaction and program flow

## How to Run
* Make sure Python 3.14 is installed
* Clone repository
* Run program

## Example Menu
1. Add Income
2. Add Expense
3. View Balance
4. View Transactions
5. Exit Program
